📒 ScrumFlow – Sprint Management System

A full-stack Agile Sprint Management System designed to manage users, tasks, and sprints efficiently. Built with Angular, ASP.NET Core Web API, and PostgreSQL, and deployed in production.

👉 By building this project, you understand real company workflow like how Scrum teams manage sprints, tasks, roles, and daily development cycles in real software companies.

---

🌐 Live Links

- 🔗 Frontend: https://scrumflowapp.onrender.com
- 🔗 Backend: https://scrumflow-app-production.up.railway.app
---

🛠️ Tech Stack

💻 Frontend

- Angular
- TypeScript
- Tailwind CSS
- Angular Material

⚙️ Backend

- ASP.NET Core Web API
- Entity Framework Core
- PostgreSQL (Railway)

☁️ Deployment

- Frontend: Render
- Backend: Railway
- Database: Railway PostgreSQL

---

✨ Features

🔐 Authentication

- User Registration & Login
- Secure session handling using cookies
- Logout functionality

👥 User Management

- View users
- Update profile
- Assign roles (Admin / Developer)

📋 Task Management

- Create tasks
- Update tasks
- Assign tasks
- View tasks (My / All)

🏃 Sprint Management

- Create sprint
- Start / End sprint
- Track sprint progress

📊 Agile Workflow

- Story points tracking
- Task prioritization
- Sprint lifecycle management

---

🚀 Deployment Notes

- Angular production build:

ng build --configuration production

- Backend uses Railway "DATABASE_URL"
- CORS enabled for:
  - http://localhost:4200
  - https://scrumflowapp.onrender.com

---

🔐 Security

- BCrypt password hashing
- HttpOnly secure cookies
- CORS protection
- Session-based authentication
